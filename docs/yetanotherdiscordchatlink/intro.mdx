# Introduction

YetAnotherDiscordChatLink is available on [Modrinth](https://modrinth.com/mod/yet-another-discord-chat-link) and [Curseforge](https://www.curseforge.com/minecraft/mc-mods/yet-another-discord-chat-link).

This mod that connects the in-game chat with a Discord channel to show chat messages, join/leave messages and server status messages.
