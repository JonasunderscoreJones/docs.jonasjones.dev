# Basic
The basic routes of the API

## GET /v1/help
Get a help message.

:::tip[**Example** Response]
<Tabs>
    <TabItem value="200" label="200">
```json
Please refer to the wiki at https://wiki.jonasjones.dev/Api/
```
    </TabItem>
    <TabItem value="503" label="503">
```json
API is down
{
    "status": 503,
    "statusText": "Service Unavailable"
}
```
    </TabItem>
</Tabs>
:::
